🔥 TỔNG QUAN PHÂN CÔNG (FINAL)
Người	Service	Vai trò chính
1	Auth Service	Xác thực
2	Content Service	AI xử lý text (CORE)
3	Media Service (Visual AI)	AI xử lý hình ảnh
4	Education Service	Quiz + Flashcard
5	Export + Workspace	Lưu trữ + xuất file
🟢 1. Người 1 — Auth Service
🎯 Bạn này làm “cổng vào hệ thống”
📦 Nhiệm vụ cụ thể
Đăng ký / đăng nhập
JWT authentication
Middleware verify token
API:
POST /auth/register
POST /auth/login
GET /auth/me
🎯 Deliverables
Login chạy được
Token dùng được cho tất cả service khác

🎯 Đây là “não AI”
📦 Nhiệm vụ
1. Kiểm duyệt
/content/moderate
2. Làm mượt nội dung
/content/enhance
3. Trích xuất keyword
/content/keywords
4. Sinh outline (slide/comic)
/content/outline
5. ❗ UPDATE THEO REQUIREMENT MỚI

👉 Xử lý user edit

POST /content/regenerate

👉 Input:

{
  "slides": [...],
  "style": "storytelling",
  "tone": "dramatic"
}

👉 Output:

nội dung mới theo ý user
🎯 Deliverables
AI generate được:
slide outline
comic script
Có thể regenerate khi user chỉnh
🔵 3. Người 3 — Media Service (Visual AI) ⭐
🎯 Không còn là “search ảnh” nữa

👉 Đây là:
🔥 AI hiểu hình ảnh theo context + style

📦 Nhiệm vụ
1. Generate keyword thông minh

(từ content)

2. Search Wikimedia
3. Filter:
chất lượng
license
đúng bối cảnh
4. ❗ QUAN TRỌNG NHẤT
POST /media/generate-assets

👉 Input:

{
  "scenes": [...],
  "style": "comic",
  "tone": "funny"
}

👉 Output:

{
  "images": [...]
}
5. Regenerate ảnh khi user không thích
POST /media/regenerate-image
🎯 Deliverables
Ảnh phù hợp:
nội dung
style (comic / serious)
Có thể đổi lại ảnh theo user
